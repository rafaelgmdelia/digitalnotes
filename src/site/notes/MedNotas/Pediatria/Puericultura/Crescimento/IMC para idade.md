# IMC para idade
| DP | < 5 anos | > 5 anos |
| ---- | ---- | ---- |
| < -3 | magreza acentuada | magreza acentuada |
| -3 a -2 | magreza | magreza |
| -2 a +1 | adequado | adequado |
| +1 a +2 | risco de sobrepeso | sobrepeso |
| +2 a +3 | sobrepeso | obesidade |
| > +3 | obesidade | obesidade grave |
