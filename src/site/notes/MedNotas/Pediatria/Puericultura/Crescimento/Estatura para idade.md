# Estatura para idade
- > +2 -> estatura adequada / alta estatura
- -2 a +2 -> estatura adequada
- -2 a -3 -> baixa estatura
- < -3 -> muita baixa estatura
- ## Ritmo de crescimento
	- Ao nascimento -> ~50cm
	- 1º ano -> + 25cm
	- 2º ano -> +10-12cm
	- 3-4º ano -> +7,5cm/ano
	- 4 anos - puberdade -> ~5cm/ano
	- Puberdade -> estirão puberal
