- Ausculta ausente
- Jugulares túrgidas
- Percussão timpânica
- Desvio de traqueia contralateral
- Instabilidade hemodinâmica -> choque obstrutivo
**Diagnóstico clínico**

# Exames complementares
- RX de Tórax
- USG

# Tratamento
## Punção de alívio
- 5º espaço intercostal
- Linha axilar média ou anterior

## Drenagem de tórax
- 5º espaço intercostal
- Linha axilar média
- RX de Tórax para confirmar posicionamento
